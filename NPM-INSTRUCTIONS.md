### Update on npm

Do changes.  
Make sure to update README.md to next NPM version.  

run `npm build`  
run `npm dist`  

Push changes on git locally.  

bump NPM version, e.g.  
`npm version patch -m "Upgrade for reasons"`

Push git to remote.  
Make sure to push tags to remote as well.  

Publish to NPM.  
`npm publish`
